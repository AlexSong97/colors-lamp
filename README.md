# COLORS LAMP Web Application

## Description
This project is a full-stack web application developed using the LAMP stack (Linux, Apache, MySQL, PHP).  
The application allows users to register, log in, and manage a personalized list of colors.

## Technologies Used
- Linux (DigitalOcean droplet)
- Apache Web Server
- MySQL Database
- PHP (Backend API)
- HTML, CSS, JavaScript (Frontend)

## Project Structure
- api/ : PHP backend API endpoints
- public/ : Frontend files and assets


## Setup Instructions
1. Install a LAMP stack (Linux, Apache, MySQL, PHP)
2. Clone this repository
3. Import the database tables:
   - Users
   - Colors
   - Contacts
4. Update database credentials inside the PHP API files.
5. Place the project inside `/var/www/html`
6. Open the browser and access `index.html`

## Features
- User login system
- Add colors to personal list
- Search colors
- REST-style API endpoints
